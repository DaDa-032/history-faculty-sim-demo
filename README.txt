# 历史学院青年教师模拟器 Demo  
*History Faculty Tenure-Track Simulator (Demo)*

这是一个**文字模拟类小游戏 Demo**，模拟一名历史学青年教师从博士毕业、入职高校，到面对“非升即走”制度、努力获得长聘教职的全过程。

游戏采用浏览器运行方式，无需安装任何额外软件，可直接在线游玩。

## 🎮 游戏简介

你将扮演一名刚刚博士毕业的历史学研究者，入职一所高校，在有限的时间内：

- 上课、备课、指导学生  
- 写论文、投稿、退稿、再投稿  
- 申请科研项目、等待结果  
- 应对各种（不太严肃但也不完全虚构的）随机事件  
- 在制度变化与个人状态波动中，努力活到“长聘”

这是一个 **制度 + 个人选择 + 运气** 共同作用的模拟游戏。

---

## 🧠 核心机制（简要）

- **时间结构**：  
  - 每年 4 个学期（第一学期 / 寒假 / 第二学期 / 暑假）
  - 每个学期分为「事件阶段」和「行动阶段」

- **个人状态**（0–1000）：
  - 心态 / 精力 / 时间 / 灵感

- **技能系统**：
  - 天赋 / 勤奋 / 社交 / 运气（1–6）
  - 技能影响消耗与收益（高技能 → 消耗更低、收益更高）

- **成果系统**：
  - C 刊 / 一流 / 顶刊论文
  - 国家级 / 省部级科研项目
  - （隐藏）领导满意度

- **非升即走考核**：
  - 教学要求
  - 科研成果要求
  - 项目要求
  - 可能发生的隐藏制度变化（不提前告知）

---

## 🎲 关于随机事件

- 随机事件并非完全随机  
- **运气值越高，越容易遇到正面事件**
- 每学期有固定数量的随机事件，必须全部处理完才能进入行动阶段
- 某些制度性变化为**隐藏事件**，不会提前告知触发概率

---

## 🏁 胜负条件

- 在规定年限内满足全部考核要求  
- 并且满足隐藏条件（如行政评价）  
- 否则视为考核失败，游戏结束

> 注：失败时，系统会告知失败原因。
